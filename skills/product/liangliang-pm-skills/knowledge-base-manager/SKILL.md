# 知识库管家

## 触发条件

1. **自动触发**：我输出完PRD后，主动询问「是否存入知识库？」
2. **用户指令**：用户说「存入知识库」「存知识库」「保存到知识库」
3. **检索指令**：用户说「查一下之前的需求」「找历史需求」「类似的需求做过吗」
4. **索引更新**：存入文档后自动更新MEMORY.md索引

## 核心能力

### 1. PRD存储询问（自动触发）

每次输出PRD后，自动询问：
```
PRD输出完毕，是否存入知识库？
（存入后可在后续需求中检索参考）
```

用户回答「是/存/好」→ 执行存储流程
用户回答「否/不存/不用」→ 跳过，记录本次不存

### 2. 文件名查重

存储前检测 `knowledge_base/` 目录：
- 检测到同名文件 → 询问「是否覆盖？」或「重命名？」
- 无重名 → 直接存入

**询问格式：**
```
检测到同名文件：PRD_有成报销_XXX_v1.0.md
[1] 覆盖原文件
[2] 重命名（建议：PRD_有成报销_XXX_v1.1.md）
[3] 取消存储
请选择：
```

### 3. 自动索引更新

存入后自动更新 `MEMORY.md` 知识库索引：
```markdown
### 新增文档
- **PRD_有成报销_XXX_v1.0.md**
  - 日期：2026-04-01
  - 核心：简要描述
  - 关联：相关需求链接
```

### 4. 历史需求检索（为其他Skill提供服务）

**为 screenshot-prd-analyzer 提供相似需求检索：**
```
输入：截图识别出的功能模块（如「预算报表」「调整金额」）
检索：knowledge_base/ + memory/ 中匹配的需求
输出：相似需求列表（相似度>80%）

示例输出：
检测到相似历史需求：
- 2026-04-01 预算报表调整金额下探预算申请单（相似度95%）
- 2026-03-22 报表优化20230717（相似度40%）

推荐参考：2026-04-01 预算报表调整金额下探
- 核心方案：仅勾选预算申请时调整金额可点击
- 数据关系：一对多
- 下探字段：10个标准字段
```

**为 prd-clarification-helper 提供历史方案检索：**
```
输入：需求类型（委托授权/审批流变更/状态流转）
检索：历史上同类复杂需求的处理方案
输出：历史方案清单 + 选择依据

示例输出：
历史上同类复杂需求：

【委托授权系列】（4个需求）
- 20230518 委托授权增加权限设置
- 20230823 审批流条件增加委托授权提单
- 20231130 委托提单单据展示控制
- 20260322 委托授权已审批单据可见性

共同方案特征：
- 均采用轻量方案（不重新匹配审批流）
- 核心原因：存量逻辑复杂，影响范围广
- 方案选择依据：业务复杂度 > 长期收益

建议本次参考该系列方案。
```

**关键词检索（用户主动查询）**
```
用户：查一下之前关于「委托授权」的需求
我：检索 knowledge_base/ 找到3个相关文档：
1. 审批流条件增加委托授权提单（20230823）
2. 委托提单数据同步设置（20231130）
3. 委托授权已审批单据可见性（20260322）
```

**相似需求推荐**
```
当前需求：驳回单据修改费用类型
相似历史需求：
- 费用标准规则继承字段功能（涉及字段编辑权限）
- 委托授权系列（涉及单据可见性控制）
是否查看参考？
```

## 工作流

### 存储流程
```
PRD输出完成 / 用户说「存入知识库」
  ↓
询问「是否存入知识库？」（如非用户主动触发）
  ↓
用户确认
  ↓
检测文件名重复
  ├─ 有重复 → 询问覆盖/重命名/取消
  └─ 无重复 → 直接存入
  ↓
更新MEMORY.md索引
  ↓
确认完成
```

### 检索流程
```
用户询问历史需求
  ↓
解析关键词（模块/功能/时间）
  ↓
检索knowledge_base/目录
  ↓
输出匹配结果列表
  ├─ 精确匹配 → 展示文档信息
  └─ 模糊匹配 → 推荐相似文档
  ↓
用户选择查看 → 读取文档内容
```

## 文档命名规范

建议文件名格式：
```
PRD_产品名_功能名_版本.md
```

示例：
- `PRD_有成报销_审批节点超时小时单位_v1.0.md`
- `PRD_有成报销_企业微信分享升级_v1.0.md`

## 索引格式

在 `MEMORY.md` 中维护：
```markdown
## Knowledge Base

### 2026年4月
- **PRD_有成报销_审批节点超时小时单位_v1.0.md**
  - 日期：2026-04-01
  - 核心：超时设置支持天/小时单位切换
  - 状态：已存知识库

### 2026年3月
- **有成报销_完整需求汇总_448文档合并.md**
  - 日期：2026-03-22
  - 核心：448个历史需求汇总
  - 规模：35,852行
```
